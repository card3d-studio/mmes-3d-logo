# Morehead Montessori 3D Logo

Interactive WebGL logo for **Morehead Montessori** built using **Three.js** and **Vite**.

This project renders a responsive, accessible 3D version of the school's logo with:

- rotation controls (mouse, touch, or keyboard)
- transparent background for embedding on websites
- reduced‑motion support
- static fallback image for non‑WebGL environments

The project is designed to be embedded into a school website header or landing page.

---

## Technology

- Three.js
- Vite
- WebGL
- GitHub Pages

---

## Development

Install dependencies:

npm install

Run the development server:

npm run dev

Build the production version:

npm run build

Preview the production build locally:

npm run preview

---

## Accessibility

The logo includes:

- keyboard rotation using arrow keys
- reduced‑motion fallback for users who disable animation
- screen‑reader descriptions
- static image fallback when WebGL is unavailable

---

## Deployment

The project is configured to deploy automatically to **GitHub Pages** using GitHub Actions.
